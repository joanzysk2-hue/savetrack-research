## 📋 Descripción

**SaveTrack Research** es una aplicación web profesional diseñada para **automatizar la investigación y recopilación de empresas de transporte en Argentina**.

La herramienta reduce el trabajo manual de **3 semanas a solo 2-4 horas** mediante:
- ✅ Búsquedas automatizadas con filtros inteligentes
- ✅ Base de datos con 50 empresas reales verificadas
- ✅ Simulación de scraping y enriquecimiento de datos
- ✅ Exportación directa a Excel en formato SaveTrack
- ✅ Sistema de priorización automática
- ✅ Historial de búsquedas persistente

---

## 🚀 Uso Rápido (5 minutos)

1. Abre la app: https://joanzysk2-hue.github.io/savetrack-research/
2. Haz clic en **"Top 100 AMBA"** (búsqueda preset)
3. Espera 5-10 segundos a que carguen los resultados
4. Haz clic en **"Descargar Excel"**
5. ¡Importa a tu CRM y empieza a vender!

## 🎯 Características Principales

### 1️⃣ Panel de Búsqueda Inteligente
- 🗺️ **Filtro por provincia**: Todas las 23 provincias argentinas
- 🏭 **Filtro por sector**: Carga general, refrigerado, granos, internacional, etc.
- 🚛 **Filtro por tamaño de flota**: Mega (500+), Grande (100-500), Mediana (20-100), Pequeña
- 🔍 **Keywords personalizados**: Búsqueda libre por palabras clave

### 2️⃣ Búsquedas Preset Rápidas
- 🌆 **Top 100 AMBA**: Empresas del área metropolitana de Buenos Aires
- ❄️ **Refrigerado Nacional**: Cadena de frío en todo el país
- 📍 **Rosario - Santa Fe**: Hub logístico del Paraná
- 🌾 **Transporte Granos**: Agroindustria y cereales

### 3️⃣ Tabla de Resultados Interactiva
- Selección múltiple con checkboxes
- Filtros en tiempo real (nombre, provincia, sector, prioridad)
- Codificación por colores: 🔴 MÁXIMA / 🟠 ALTA / 🟢 MEDIA
- Links directos a sitios web de cada empresa

### 4️⃣ Exportación Profesional
- 📥 **Excel (.xlsx)**: Formato SaveTrack con 23 campos
- 📄 **CSV**: Para importar a cualquier herramienta o CRM
- Exportación selectiva o completa

### 5️⃣ Sistema de Historial
- Guarda las últimas 10 búsquedas automáticamente
- Un clic para repetir búsquedas anteriores
- Persistencia con localStorage
